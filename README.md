# Weather Analyzer

Weather Analyzer is a Python library for processing and visualizing weather data.  
It helps developers quickly clean datasets, run statistical models, and generate charts.

## Features
- Import CSV or JSON weather datasets
- Calculate averages, anomalies, and trends
- Generate line charts and heatmaps

## Installation
```bash
pip install weather-analyzer

from weather_analyzer import Analyzer

data = Analyzer("weather.csv")
data.plot_trends()
